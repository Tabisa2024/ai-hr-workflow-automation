# 🤖 AI HR Workflow Automation System

## 📌 Overview
This project is an AI-powered automation system designed to reduce manual work in HR processes.

It automates tasks such as leave requests, approvals, reporting, and employee queries using workflow automation and AI.

---

## 🚀 Problem
In many organizations, HR processes are manual:
- Employees submit requests manually
- Approvals take too long
- Reports are generated manually
- High risk of human error

---

## 💡 Solution
This system automates HR workflows by:
- Capturing requests automatically
- Triggering approval workflows
- Using AI to process queries
- Generating reports automatically

---

## 🛠 Tech Stack
- Python
- OpenAI API
- JSON
- Workflow logic

---

## 🏗 Architecture
System flow:

User → AI Processing → Workflow Engine → Output
